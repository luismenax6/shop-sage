# Order Changes and Cancellations

## Changing an order
You can change the shipping address, add items, or update quantities **while the order is still in "processing" status**. Ask the assistant to update your order with the order number and the change you want. Once an order moves to "shipped," it can no longer be modified.

## Cancelling an order
Orders in "processing" status can be cancelled for a full refund. If the order has already shipped, you will need to refuse delivery or start a return once it arrives. Cancellations are confirmed by email.

## Out-of-stock items
If an item sells out after you order, we will notify you and offer a comparable replacement, a backorder, or a refund for that item. The rest of your order ships on schedule.

## Combining orders
We are unable to combine separate orders into one shipment once they are placed. If you need items together, cancel and reorder while both orders are still processing.

## Address accuracy
Please double-check your shipping address at checkout. We are not responsible for orders delivered to an incorrect address provided by the customer, though the assistant can help you start a carrier trace.
